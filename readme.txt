Enables easy blog post link submission to eyeCanGo Travel Blog Gallery. Use 'Submit to eyeCanGo' from post edit view for published posts.

== Description ==

eyeCanGo is a travel blog gallery that people use to explore firsthand experiences at their destinations and plan their ultimate vacation.
Use this plugin if you are a travel blogger and would like to link up your posts to eyeCanGo.

Once installed and activated, this plugin will add a new admin bar menu item to the top called 'Submit to eyeCanGo'.
It is only visible when you are on the edit page of a post that you have already published. Activating the menu item
opens a new tab that takes you to the submission page of the eyeCanGo website. Submitting content to eyeCanGo requires signing in so you might
be promted to sign in (or join if you are not yet a member).

== Installation ==

1. Upload `ecgsubmit` folder to the `/wp-content/plugins/` directory (or install from the zip archive of the same name).
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= Do I need to re-submit a post if I change it? =

It is not necessary in most cases. However you may want to re-submit your post if you have changed the title or short description of the post or
you want to adjust the photo or tags that you submitted previously.
